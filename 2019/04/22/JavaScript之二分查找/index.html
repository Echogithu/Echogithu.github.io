JavaScript之二分查找

发表于 2019-04-22 | 分类于 算法

如果查找的数据有序的，二分查找算法更加高效。

算法理解

二分搜索算法有些类似于玩猜数字游戏，这个数字位于1~100之间，要猜的数字是由出题人决定的。我们每说一个数字，出题人就会说大了，小了还是对了。所以，第一次猜50是最佳策略，如果猜大了，我们就猜25，如果猜小了，我们就猜75。以此类推，每一次猜测，我们都选择最大值和最小值的中间值。我们把这个策略实现为二分搜索算法。

二分搜索算法需要保证被搜索的数据是有序的，因此我们先用排序算法把数组的数据排序。

算法描述

  1. 选择中间值
  2. 如果选择的值是待搜索的值，算法结束并返回。
  3. 如果待搜索值比选中值要小，则返回步骤1，并在选中值左边的子数组中寻找。
  4. 如果待搜索值比选中值要大，则返回步骤1，并在选中值右边的子数组中寻找。
