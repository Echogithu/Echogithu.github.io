成员访问

在 ECMAScript 中成员的访问主要有两种方式：
1.“点” 表示法
2.“[]” 方括号表示法

它们的区别是 “点” 表示法更通用，更直观，更快捷。但缺点则是无法访问含有特殊字符、关键字、保留字的成员名。

此时，括号表示法的强大就体现了出来。

Array类型

创建数组的方式
  1. 使用Array的构造函数

  • 如果事先知道数组的长度可以传一个数字，并且该数字会变成该数组length属性的值。

    • 也可以向数组传入应该包含的项。

  • 使用Array构造函数时，也可以省略new操作符。

  1. 创建数组的第二种基本方式是数组字面量表示法。

  • 当设置的索引大于数组现有的项数的时，数组就会自动增加到该索引加一的长度

  • 数组的length可读可写，通过设置length属性可以从数组的末尾移除项或者向数组中添加项。

检测数组
  1. 对于一个网页或者一个全局作用域而言，使用instanceof即可。

缺点：
如果使用iframe形式嵌入网页，那么至少存在两个以上的全局执行环境，从而存在两个以上不同版本的Array构造函数。这样第一个框架与第二个框架分别具有不同的构造函数。

  1. 也可以利用Object.prototype.toString来判断。

  2. Array.isArray()

转换方法

数组的三个转换方法：

  1. toString
    toString() 与 toLocaleString() 方法会将数组元素转换为字符串，并以“逗号”作为默认的分隔符进行拼接返回。

  2. toLocaleString
    与toString类似，不过是调用数组的每一项的toLocaleString方法

  3. valueOf
    返回数组实例对象本身。
栈方法

数组提供的push和pop方法可以让其像栈数据结构一样，完成(LIFO)后进先出。

  • ECMAScript 实现栈的行为主要借助 push() 、pop() 等方法：
  1. push() ：向数组的尾部追加新的数组元素。返回更新后的数组长度。
  2. pop() ：删除数组尾部的最后一项，返回被删除的数组元素。

通过使用 pop() 、push() 来模拟“栈”的结构。那么栈头的位置就是数组的尾部，因为这两个方法都是对数组的尾部进行操作。

队列方法

数组提供的shift和push方法可以让其像队列数据结构一样，完成(FIFO)先进先出。

使用数组来模拟队列，需要借助 shift() 、push() 等方法。

shift() ：删除数组第一个元素，并返回被删除的数组元素。

也可以使用unshift和pop实现相反方向的队列结构。

实现反向队列则需要借助 unshift() 与 pop()。

unshift() ：向数组的首部添加元素，并返回添加后的数组长度。

重排序方法

  1. reverse() 方法会对原数组进行颠倒重排，并永远改变原数组中数组元素的顺序。

  2. sort() 方法默认升序排序，并且会自动将数组元素转换为字符串，然后按照 ASCII 码大小进行排序。

如果只是单单的将数组元素转换为字符进行大小比较，必然无法满足一些更为复杂的使用场景，因此 sort() 方法也接收一个回调函数，来自定义排序的行为：

操作方法
concat()

基于当前的数组中的所有项创建一个新的数组，具体来说就是先创建当前数组的一个副本，然后将接收到的参数添加到这个副本的末尾，最后返回新构建的数组。

  1. 如果没有传参数，只是简单地对当前数组的拷贝。
  2. 如果传递的是一个或者多个数组，则该方法会将这些数组中的所有项都添加到该数组中。
  3. 如果传递的不是数组，这些值就会简单地添加到数组的末尾。
slice()

slice() 方法返回一个从开始到结束（不包括结束）选择的数组的一部分浅拷贝到一个新数组对象。原始数组不会被修改。

  1. 如果没有传参数则表示对当前数组的一个浅拷贝。
  2. 如果传递了一个参数则返回从该参数指定的位置开始到当前数组末尾所有的项。
  3. 如果传递了两个参数则返回起始位置到结束位置的所有的项(不包含结束位置)

splice()

solice()方法可以实现删除、插入、替换等功能，且直接改变原数组的内容。主要用途是向数组的中部插入项。

  • 删除：可以删除任意数量的项，只需指定2个参数：起始位置，删除项数
  • 插入：可以向指定位置插入任意数量的项，只需指定3个参数：起始位置，0，插入项
  • 替换：可以向指定位置插入任意数量的项，且同时删除任意数量的项，只需指定3个参数：起始位置，删除项数，插入项
位置方法

两个位置方法：

  1. indexOf() ：从数组开头开始向后查找
  2. lastIndexOf() ：从数组末尾开始向前查找。
  • 参数：要查找的项 和 表示查找起点位置的索引。
  • 返回值：要查找项的位置，没找到则返回-1。
迭代方法

5个迭代方法：

  1. every():
    对数组中的每一个项运行给定的函数，如果该数组的每一项都返回true，那么结果也返回true
  2. some() ：
    对数组中的每一项运行给定的函数，如果该项对任一项返回true，就返回true
  3. forEach():
    对数组中的每一项运行给定的函数，该函数没有返回值
  4. map() :
    对数组中的每一项运行给定的函数，返回每次调用函数的结果组成的数组
  5. fiflter() :
    对数组的每一项运行给定的函数，返回该函数返回true的项组成的数组
  • 接收两个参数：
    • 要在每一项上运行的函数
    • 运行该函数的作用域对象：this
  • 传入的函数会接收三个值：
    • 数组项的值
    • 该项在数组中的位置
    • 数组对象本身
缩小方法

两个缩小数组的方法：

  1. reduce() : 从数组第一项开始，逐个遍历到最后。
  2. reduceRight() ：从数组最后一项开始，向前遍历到第一项。
